## Description

For this task, I created a digital literacy awareness infographic using Canva. The design visually presents five key areas: the definition of digital literacy, safe internet practices, maintaining a professional online presence, email etiquette, and useful digital tools for students. I found it challenging yet interesting to condense important information into a clean, engaging layout without overcrowding the space. This exercise helped me understand how to create effective visual resources that can quickly educate peers on essential digital skills.
